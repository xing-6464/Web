## 高级特性

### v-model

- Vue2中定义自定义组件`v-model`，先在`model`中`prop`定义双向绑定数值名字，`event`设置触发事件名字


```javascript
Vue.component('base-input', {
  model: {
    prop: 'text',
    event: 'change'
  },
  props: {
    text: String,
    default () {
      return ''
    }
  },
  template: `
    <input
      type="text"
      :value="text"
      @input="$emit('change', $event.target.value)"
    >
  `
})
```

- 现在在这个组件上自定义组件`v-model`的时候
```html
<base-input v-model="text"></base-input>
```
